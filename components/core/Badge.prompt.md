**Badge** — compact pill for categories, prices, popularity, and status.

```jsx
<Badge tone="rose">Popular</Badge>
<Badge tone="gold">40,00 €</Badge>
<Badge tone="neutral">1h 30min</Badge>
```

Tones: `rose` (default soft), `gold` (prices/premium), `neutral`, `solid`, `outline`. Sizes `sm | md`. Always renders tracked uppercase.
