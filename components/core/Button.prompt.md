**Button** — the primary interactive control; use for booking CTAs, form submits, and navigation actions.

```jsx
<Button variant="primary" size="lg" href="#book">Reservar cita</Button>
<Button variant="outline">Ver servicios</Button>
<Button variant="gold" size="sm">Oferta</Button>
```

Variants: `primary` (rose, default), `gold` (premium/offers), `outline`, `ghost`, `dark` (on light footers). Sizes `sm | md | lg`. Labels render in tracked uppercase sans automatically. Set `href` to render an anchor, `full` to stretch.
