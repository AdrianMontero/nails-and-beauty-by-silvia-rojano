**Card** — soft rounded surface; base container for services, reviews, feature blocks.

```jsx
<Card tone="white" hover pad="lg">…</Card>
<Card tone="blush">…</Card>
```

Tones: `white` (default), `blush`, `deep`, `dark`. Set `hover` for a lift on mouse-over. `pad` controls inner spacing (`sm | md | lg | xl`).
